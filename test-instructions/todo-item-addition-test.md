# Todoアイテム追加テスト

## 目的
新しいTodoアイテムを追加できることを確認

## 手順
1. 入力フィールドに「テストタスク1」を入力
2. 「追加」ボタンをクリック
3. リストに「テストタスク1」が表示されることを確認
4. 入力フィールドがクリアされることを確認
5. 入力フィールドに「テストタスク2」を入力
6. 「追加」ボタンをクリック
7. リストに「テストタスク1」と「テストタスク2」の両方が表示されることを確認

## 期待結果
- 新しいタスクがリストに追加される
- 各タスクにチェックボックスと削除ボタンが表示される
- 入力フィールドが自動的にクリアされる
